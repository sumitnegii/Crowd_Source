# CrowdEmergencyResponse
CrowdEmergencyResponse is a simple web application that helps users report emergencies and see real-time updates. The goal of this project is to make it easier for people to get quick help by showing nearby incidents, emergency contacts, and live information.

Features

Report an emergency with location

View live emergency incidents

Basic real-time feed using WebSockets/Firebase

Simple and clean UI

Easy to use for general users

Tech Stack

Frontend: React.js

Backend: Node.js / Firebase

Database: Firestore / Realtime Database

Other Tools: WebSockets, Google Maps (optional)

How to Run

Clone the project

git clone <repo-url>


Install dependencies

npm install


Start the project

npm start

Status

The project is still in progress. More features like CrowdAI assistant, verified responders, and live dashboards will be added step by step.
